    Backup & Sync

    Data lives in localStorage. Export to JSON for backups or to paste into MEMORY.md.

    1. To sync with MEMORY.md or a JSON file, click Copy JSON.
    2. Paste into MEMORY.md or overwrite dashboard-data.json in the workspace for file persistence.
    3. Use Restore JSON to pull that file back into the PWA.

    Usage tips

    • Install via the button above for an app-like feel.
    • Use the Data Export after major updates.
    • Embed OpenClaw chats below to keep conversations handy.
    • Weather uses Open-Meteo (free, no key).
